# TUI Notes — Comprehensive Implementation Plan

> **For Hermes:** This plan is ready for task-by-task execution. Each task is bite-sized (2–5 min) with exact paths, complete code, and verification steps.

**Goal:** Build a lightweight, terminal-aesthetic markdown notes app using Tauri v2 (Rust) + vanilla JS frontend. The file system is the database. Zero visual bloat, instant keyboard/mouse response, crash-safe plain-text storage.

**Architecture:** Tauri v2 shell with a Rust backend exposing 4 IPC commands. Frontend is a single HTML file with vanilla JS and CSS. No frameworks, no build step. State = 3 variables (`activeNotePath`, `fileTree`, `isDirty`).

**Tech Stack:** Tauri v2, Rust, Vanilla JS, HTML5 drag-and-drop, `marked` (or `micromark`) for preview, single monospace font stack.

---

## Validation & Key Decisions

Your blueprint is solid. Below are the decisions I’ve locked in, plus a few gaps I’ve filled.

| Decision | Choice | Reason |
|----------|--------|--------|
| **Framework** | Vanilla JS (no Svelte) | Keeps build step at zero; Svelte adds complexity for marginal gain in a 2-panel app. |
| **Markdown parser** | `marked` (single-file, ~30KB) | `micromark` is modular but heavier to wire up; `marked` is drop-in. |
| **File tree refresh** | Poll every 2s + manual refresh | Tauri v2 has `fs.watch`, but polling is simpler and more reliable for v1. We can add watch later. |
| **Auto-save** | Debounced 1s after typing stops | Prevents data loss without blocking the UI. Manual `Ctrl+S` also works. |
| **New note naming** | `untitled-{timestamp}.md` in root | Avoids modal dialogs; user renames by editing the file directly in the tree. |
| **Folder picker** | Native Tauri dialog on first run | No config file needed; path stored in `localStorage`. |
| **Mobile (Android)** | Deferred | Tauri v2 supports it, but desktop-first keeps scope tight. |

### Gaps Filled

1. **Window chrome:** Use Tauri’s built-in decorations (no custom titlebar) to keep scope minimal.
2. **Error handling:** All Rust commands return `Result<T, String>`. Frontend shows a red banner at the top for 3s on error.
3. **Empty state:** If no note is open, right panel shows `// no note selected` in the same monospace style.
4. **Binary/large files:** Rust backend will skip non-`.md` files and files > 1MB from the tree to prevent accidents.
5. **Line endings:** Rust backend normalizes to `\n` on read; writes back exactly as received.

```rust
use serde::{Deserialize, Serialize};
use std::fs;
use std::path::Path;
use walkdir::WalkDir;

#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct FileNode {
    pub name: String,
    pub path: String,
    pub is_dir: bool,
    pub children: Vec<FileNode>,
}

#[tauri::command]
pub fn scan_directory(path: String) -> Result<Vec<FileNode>, String> {
    let root = Path::new(&path);
    if !root.exists() {
        return Err(format!("Directory does not exist: {}", path));
    }
    if !root.is_dir() {
        return Err(format!("Path is not a directory: {}", path));
    }

    let mut nodes = Vec::new();
    for entry in WalkDir::new(root)
        .min_depth(1)
        .max_depth(10)
        .follow_links(false)
        .into_iter()
        .filter_map(|e| e.ok())
    {
        let path = entry.path();
        let name = path.file_name()
            .and_then(|n| n.to_str())
            .unwrap_or("unknown")
            .to_string();
        let is_dir = path.is_dir();
        let rel_path = path.strip_prefix(root)
            .map_err(|e| e.to_string())?
            .to_string_lossy()
            .to_string();

        // Skip non-md files and hidden files
        if !is_dir {
            if !name.ends_with(".md") {
                continue;
            }
            if name.starts_with('.') {
                continue;
            }
            // Skip files > 1MB
            if let Ok(meta) = path.metadata() {
                if meta.len() > 1_048_576 {
                    continue;
                }
            }
        }

        nodes.push(FileNode {
            name,
            path: rel_path,
            is_dir,
            children: Vec::new(),
        });
    }

    // Build tree structure
    let tree = build_tree(nodes, root);
    Ok(tree)
}

fn build_tree(flat: Vec<FileNode>, root: &Path) -> Vec<FileNode> {
    let mut tree = Vec::new();
    let mut dirs: Vec<&FileNode> = flat.iter().filter(|n| n.is_dir).collect();
    dirs.sort_by(|a, b| a.path.cmp(&b.path));

    for dir in dirs {
        let node = dir.clone();
        insert_node(&mut tree, node, root);
    }

    let mut files: Vec<&FileNode> = flat.iter().filter(|n| !n.is_dir).collect();
    files.sort_by(|a, b| a.name.cmp(&b.name));

    for file in files {
        let node = file.clone();
        insert_node(&mut tree, node, root);
    }

    tree
}

fn insert_tree(tree: &mut Vec<FileNode>, node: FileNode, root: &Path) {
    let parent_path = Path::new(&node.path)
        .parent()
        .map(|p| p.to_string_lossy().to_string());

    match parent_path {
        Some(pp) if !pp.is_empty() => {
            if let Some(parent) = find_node_mut(tree, &pp) {
                parent.children.push(node);
            } else {
                tree.push(node);
            }
        }
        _ => tree.push(node),
    }
}

fn find_node_mut<'a>(tree: &'a mut [FileNode], path: &str) -> Option<&'a mut FileNode> {
    for node in tree.iter_mut() {
        if node.path == path {
            return Some(node);
        }
        if node.is_dir {
            if let Some(found) = find_node_mut(&mut node.children, path) {
                return Some(found);
            }
        }
    }
    None
}
```

```rust
#[tauri::command]
pub fn read_note(path: String) -> Result<String, String> {
    fs::read_to_string(&path)
        .map(|s| s.replace("\r\n", "\n"))
        .map_err(|e| format!("Failed to read {}: {}", path, e))
}

#[tauri::command]
pub fn save_note(path: String, content: String) -> Result<(), String> {
    fs::write(&path, content)
        .map_err(|e| format!("Failed to save {}: {}", path, e))
}

#[tauri::command]
pub fn move_node(old_path: String, new_path: String) -> Result<(), String> {
    fs::rename(&old_path, &new_path)
        .map_err(|e| format!("Failed to move {} to {}: {}", old_path, new_path, e))
}

#[tauri::command]
pub fn create_note(dir_path: String, name: String) -> Result<String, String> {
    let full_path = Path::new(&dir_path).join(format!("{}.md", name));
    if full_path.exists() {
        return Err(format!("Note already exists: {}", full_path.display()));
    }
    fs::write(&full_path, "")\n        .map_err(|e| format!("Failed to create {}: {}", full_path.display(), e))?;
    Ok(full_path.to_string_lossy().to_string())
}
```

**Step 3: Implement `loadTree` and `renderTree`**

```javascript
async function loadTree() {
  try {
    fileTree = await invoke('scan_directory', { path: rootPath });
    renderTree();
  } catch (e) {
    showError(e);
  }
}

function renderTree() {
  flatItems = [];
  fileTreeEl.innerHTML = '';
  renderNodes(fileTree, fileTreeEl, 0);
  updateSelection();
}

function renderNodes(nodes, container, depth) {
  for (const node of nodes) {
    const div = document.createElement('div');
    div.className = `tree-item ${node.is_dir ? 'folder' : 'note'}`;
    div.dataset.path = node.path;
    div.dataset.isDir = node.is_dir;
    div.style.paddingLeft = `${8 + depth * 16}px`;

    if (node.is_dir) {
      div.textContent = `[-] ${node.name}/`;
      div.draggable = true;
    } else {
      div.textContent = node.name.replace(/\.md$/, '');
      div.draggable = true;
    }

    div.addEventListener('click', () => {
      if (node.is_dir) {
        toggleFolder(div, node);
      } else {
        openNote(node.path);
      }
    });

    div.addEventListener('dragstart', (e) => {
      e.dataTransfer.setData('text/plain', node.path);
    });

    div.addEventListener('dragover', (e) => {
      e.preventDefault();
      div.classList.add('selected');
    });

    div.addEventListener('dragleave', () => {
      div.classList.remove('selected');
    });

    div.addEventListener('drop', async (e) => {
      e.preventDefault();
      div.classList.remove('selected');
      const srcPath = e.dataTransfer.getData('text/plain');
      const destDir = node.is_dir ? node.path : node.path.split('/').slice(0, -1).join('/');
      const fileName = srcPath.split('/').pop();
      const newPath = destDir ? `${destDir}/${fileName}` : fileName;
      try {
        await invoke('move_node', {
          oldPath: `${rootPath}/${srcPath}`,
          newPath: `${rootPath}/${newPath}`,
        });
        await loadTree();
      } catch (e) {
        showError(e);
      }
    });

    container.appendChild(div);
    flatItems.push(div);

    if (node.is_dir && node.children.length > 0) {
      const childContainer = document.createElement('div');
      childContainer.className = 'folder-children';
      container.appendChild(childContainer);
      renderNodes(node.children, childContainer, depth + 1);
    }
  }
}
```

```javascript
document.addEventListener('keydown', (e) => {
  // Ctrl+P / Cmd+P: preview
  if ((e.ctrlKey || e.metaKey) && e.key === 'p') {
    e.preventDefault();
    togglePreview();

  // Arrow keys in file navigator
  if (document.activeElement === fileTreeEl || document.activeElement === document.body) {
    if (e.key === 'ArrowDown' || e.key === 'ArrowUp') {
      e.preventDefault();
      const dir = e.key === 'ArrowDown' ? 1 : -1;
      selectedIndex = Math.max(0, Math.min(flatItems.length - 1, selectedIndex + dir));
      flatItems[selectedIndex]?.focus();
    }
    if (e.key === 'Enter') {
      const item = flatItems[selectedIndex];
      if (item && item.dataset.isDir !== 'true') {
        openNote(item.dataset.path);
      }
    }
  }
});
```

## Risks & Open Questions

1. **Drag-and-drop into subfolders:** The current `drop` handler places files into the target folder or the parent of the target note. Nested folder drops need testing.
2. **Large directories:** `walkdir` scans recursively up to depth 10. Very large directories may block the Rust thread; consider async scanning later.
3. **File watcher:** Not implemented. External changes require manual refresh (F5 or restart). We can add `notify` crate later.
4. **Mobile:** Deferred. Tauri v2 Android pipeline requires additional config (`tauri android init`).
5. **Font rendering:** `caret-shape: block` is a draft CSS property. If it doesn’t render as a block in all WebKit versions, the caret will default to a line — acceptable fallback.
